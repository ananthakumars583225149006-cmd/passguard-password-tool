# 🔐 PassGuard - Password Strength Analyzer & Generator

A powerful, real-time password security tool that analyzes password strength and generates secure passwords. Built as part of the VibeCoding Hackathon 6.0.

## 🎯 Features

| Feature | Description |
|---------|-------------|
| **Real-Time Analysis** | Checks password strength as you type |
| **Color-Coded Strength** | Weak → Fair → Good → Strong (visual feedback) |
| **Password Generator** | Creates strong 16-character passwords |
| **Copy to Clipboard** | One-click copy passwords securely |
| **Toggle Visibility** | Show/hide password text |
| **4 Security Criteria** | Length, Uppercase/Lowercase, Numbers, Special Characters |
| **100% Client-Side** | No data stored or sent anywhere |

## 🔍 Security Criteria Checked

| Criterion | Description |
|-----------|-------------|
| ✅ **Length** | At least 8 characters |
| ✅ **Case Sensitivity** | Contains both uppercase and lowercase letters |
| ✅ **Numbers** | Contains at least one number |
| ✅ **Special Characters** | Contains special characters (!@#$%^&*) |

## 📖 How to Use

### Option 1: Type Your Password
1. Type your password in the input field
2. Watch the strength meter update in real-time
3. See which criteria you've met (✅) and which you haven't (⬜)
4. Get your strength score with color-coded feedback

### Option 2: Generate Strong Password
1. Click **"Generate Strong Password"**
2. A 16-character secure password is created
3. Copy it with one click
4. Use it anywhere!

### Example Password Analysis

| Password | Score | Rating |
|----------|-------|--------|
| `password` | 25% | Weak |
| `Password123` | 50% | Fair |
| `P@ssw0rd` | 75% | Good |
| `P@ssw0rd!2024` | 100% | Strong |
